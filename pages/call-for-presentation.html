Call for Presentation

LOADays Call-for-Presentation 2015

The CFP is open until 15th of March.

How to submit your talk proposal?

Mail

Send your proposal to cfp@loadays.org with following info :

  • title of your talk
  • abstract
  • some info about yourself

Github

Using a Pull request by following the next steps :

  • Fork this site or the pelican-site repo at https://github.com/loadays/pelican-site
  • Add your proposal to the content/proposals dir. A "testtalk" is available as template (template.md).
  • Send us a Pull Request
